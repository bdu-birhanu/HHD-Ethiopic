HHD-Ethiopic: Historical Handwritten OCR Dataset for Ethiopic

Authors: Anonymous-Author

  • Historical Handwritten OCR Dataset for Ethiopic Script recognition:

    This dataset comprises approximately 80,000 historical handwritten text-line images of the Ethiopic script, each paired with its corresponding Ground-Truth (GT). It provides extensive data for training and testing recognition models. Both the images and their corresponding labels are stored in numpy format, with the images stored in their original size, allowing for more precise analysis and comparison of different recognition methods. The ground-truth sequences have a maximum length of 46 characters, and the numpy files are zero-padded. Additionally, a text-file equivalent of the ground-truth is provided. You have also the option to access the dictionary in order to reverse the integer index labels back to the text ground truth.

    The train data consists of 57,374 images of text lines, extracted from 19th and 20th century Ethiopic manuscripts, along with their corresponding ground-truth text. Both the images and ground-truth text are saved in numpy format. You can access the

    The test data consists of two sets:

    1. Test-set-I contains 6,375 text lines randomly selected from the training set, with filenames x_test_rand.npy and y_test_rand.npy for the images and ground-truth text, respectively. Additionally, a raw text file is provided as y_test_rand_text.txt.

    2. Test-set-II contains 15,935 text lines extracted from an 18th century Ethiopic manuscript set, with filenames x_test_18th.npy and y_test_18th.npy for the images and ground-truth text, respectively. A raw text file is provided as y_test_18th_text.txt.
